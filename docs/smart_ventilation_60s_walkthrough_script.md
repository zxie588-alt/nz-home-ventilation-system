# 60-Second Portfolio Walkthrough Script

This script can be used for a LinkedIn video, website walkthrough or interview screen-share. The goal is to show the project quickly without sounding like a course assignment.

## 0-10 seconds - Problem and Context

> This is my self-directed smart ventilation case study for a New Zealand residential MVHR / ERV system. I wanted to connect building-services design, local NZ context, CAD modelling and validation into one project.

Visual: portfolio project title and 3D home ventilation model.

## 10-20 seconds - HVAC Design Basis

> The base design includes a balanced normal mode of 60 L/s supply and 60 L/s extract, plus a separate 85 L/s boost target for cooking, shower or high-CO2 events. I checked duct velocities, pressure losses and a concept LGH-35 Lossnay unit selection.

Visual: airflow schedule, 2D plan and equipment selection.

## 20-30 seconds - NZ Reference Check

> I treated NZBC G4, H1 and Healthy Homes as design references, not as a compliance claim. The project clearly separates preliminary reference checks from professional consent design.

Visual: reference matrix and M-101 design review sheet.

## 30-40 seconds - SolidWorks Validation Kit

> To move beyond a static layout, I modelled a SolidWorks validation kit with a 150 mm test duct, sensor pod, pressure taps, ESP32 electronics tray, fan module and flow straightener.

Visual: SolidWorks validation kit preview and STEP download.

## 40-50 seconds - Data and Controls

> The dashboard demonstrates how CO2, humidity, airflow mode and fan power can drive normal, boost and recovery decisions. The current data is simulated, with physical sensor testing listed as future validation work.

Visual: interactive dashboard sliders and trend chart.

## 50-60 seconds - Engineering Value

> The project shows how I approach engineering work: define assumptions, model the system, size it, select equipment, identify risks, design a test method and communicate limitations clearly.

Visual: FMEA, manufacturing package and final project page.

## Short LinkedIn Caption

I upgraded my NZ residential MVHR / ERV portfolio project into a smart ventilation case study covering HVAC sizing, NZ design context, SolidWorks validation-kit modelling, commissioning planning, sensor/data workflow, FMEA and a lightweight digital-twin dashboard.
